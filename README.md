# DBPN-MindSpore

#### 介绍
DBPN的MindSpore版本

#### 软件架构
Python3.7.6 + MindSpore1.2.0

#### 使用说明
1.  下载数据集和VGG19预训练权重(https://pan.baidu.com/s/1ZhOia9ojtkJvUXQGqcM2MQ    myyu),并分别放到data/DIV2K_train_HR和vgg19文件内。
2.  训练：python main_gan.py
3.  测试：python eval_gan.py
